# Dashboard de Análise de Ecommerce

Projeto desenvolvido para análise estatística de dados de ecommerce utilizando Python, Dash e Plotly.

## Visualizações incluídas

- Histograma
- Dispersão
- Mapa de calor
- Gráfico de barras
- Gráfico de pizza
- Gráfico de densidade
- Regressão linear

## Tecnologias

- Python
- Pandas
- Plotly
- Dash
- Seaborn
- Matplotlib
